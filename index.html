Project 2

Task Management Web App

Overview
Create a task management web app that allows users to organize their daily tasks, set deadlines, and track progress. The app should feature task categories, due dates, priority levels, and a simple, intuitive interface.

Features

1. Task Creation: Users can create and categorize tasks.
2. Due Dates & Deadlines: Assign deadlines to tasks and track progress.
3. Priority Levels: Set task priorities to focus on important items.
4. Reminders: Send push or email notifications for upcoming tasks.
5. User Dashboard: Provide an overview of all tasks with status updates.
